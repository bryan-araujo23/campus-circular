# Campus Circular

Marketplace universitário de economia circular para facilitar a venda e a doação de livros, materiais acadêmicos, equipamentos e outros itens entre estudantes.

## Sobre o projeto

O Campus Circular está sendo desenvolvido como parte do desafio técnico do processo seletivo para estágio Full-Stack do Laboratório de Inovação Vortex.

A aplicação será composta por uma API RESTful e uma interface web responsiva instalável como PWA. No desktop, o sistema funcionará como uma landing page pública. No celular, oferecerá uma experiência semelhante à de um aplicativo.

## Status do projeto

Em planejamento e desenvolvimento inicial.

## Funcionalidades planejadas

- Visualização pública dos anúncios mais recentes.
- Pesquisa e filtros por categoria.
- Cadastro de itens para venda ou doação.
- Identificação local do usuário.
- Visualização dos próprios anúncios.
- Edição e exclusão de anúncios.
- Interface responsiva.
- Instalação como PWA.
- Visualização offline de dados carregados anteriormente.
